Unoriginal Goose Game

Stage 4 Advanced Game Technologies Coursework

Technology Used

  • C++
    •

    I developed the engine for this entirely original and innovative game as part of my Advanced Game Technologies module.
    The task involved implementing real-time collision detection and resolution funcitonality for AABB, sphere, and OBB collisions
    using projection, impulse, and penalty methods.
    I also implemented a basic AI state machine and pathfinding using A*.
    I explored some extra techniques such as generating level geometry from a layout defined in a config file, with the pathfinding map adapting to
    this changable level layout.
    Part 2 of the coursework saw me implementing networking features, allowing multiple players to join the game, with the server playing seeing
    these networked players moving in real time. Actual player interaction and gameplay features eluded me in the time allowed.
